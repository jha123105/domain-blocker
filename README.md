# Domain Blocker (Bettercap)

A modern GUI tool to block domains on your local network using Bettercap's ARP spoofing and HTTP/HTTPS proxy. Built with Python and CustomTkinter.

**⚠️ Educational Purpose Only:** This tool is for learning about network security and testing on your own network only. Do not use on networks without explicit permission.

## Features

- 🚦 **Start/Stop blocking** with one click
- ➕ **Add/Remove domains** to blocklist
- 🔒 **Blocks domains at DNS level** using Bettercap
- 📝 **Real-time log output**
- 🎨 **Modern dark mode GUI**

## Requirements

- Linux (Bettercap required)
- Bettercap installed (`sudo apt install bettercap` or from source)
- Root privileges (for ARP spoofing)
